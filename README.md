# andrewsuzuki/dotfiles

These are my dotfiles for Arch Linux.

## Prereqs

dependencies

* `X11`
* `rcm`
* `i3`
* `i3lock`
* `termite`
* `xbindkeys`
* `dunst`
* `git`
* `neovim`
* `udiskie`
* `autorandr`
* `gnome-keyring`
* `qt5ct`
* `rofi`
* `borg`
* `polybar`
* `PulseAudio`
* `pactl`
* `playerctl`
* `z` (in /usr/share/z/z.sh)
* `oh-my-zsh` (in ~/.oh-my-zsh)
* ...more

## Installation

1. Touch `~/.secrets.sh`
2. Run `rcup -v`
